CobolWorks

CobolWorks es un entorno de desarrollo integrado (IDE) de alto rendimiento, diseñado específicamente para desarrolladores de COBOL. Construido con C++ y Qt6, su objetivo es revitalizar el mantenimiento y la migración de sistemas legacy proporcionando herramientas de nivel profesional en una interfaz multiplataforma optimizada.

Características Destacadas

  • Análisis de Flujo Interactivo: Visualización en tiempo real de los árboles de ejecución para sentencias `PERFORM` y `GO TO`, facilitando el seguimiento de lógica compleja.
  • Migración Impulsada por IA: Integración nativa con la API de Gemini para traducir automáticamente rutinas COBOL a lenguajes modernos como Python y Java.
  • Depuración Visual: Interfaz gráfica para GDB/LLDB que permite establecer puntos de interrupción con un clic e inspeccionar el estado de forma visual.
  • Navegación Inteligente de Copybooks: Funcionalidad de "saltar a definición" y apertura automática de archivos `COPY` para agilizar la exploración de grandes bases de código.
  • Compatibilidad Multiplataforma: Ejecución nativa en Windows y Linux sin necesidad de configuraciones manuales complejas.
  • Open Source y Profesional: Código 100% abierto con binarios oficiales firmados para entornos profesionales.
